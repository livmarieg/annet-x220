Annet x220
==========
**System configuration for nixOS on the x220**

Features
--------
* Network Automation
* Security Automation
* Service Automaton
* User Automation
* Hardware Automation
* Package Automation
* Misc Config Automation
  
Curling (needs to be updated)
-----------------------------
